# Ika Ins

### 这是什么?

一个基于 *Flask* 与 *Bootstrap* 的**匿名版**。已经实现了以下功能: (截止至 V1.1.0)

- 匿名发布帖子
- 基于 ip 地址的 ID 生成及校验 (可校验，不可追溯)
- 贴图功能
